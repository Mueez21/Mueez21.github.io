Research

A Lightweight CBAM-Enhanced EfficientNetB2 Architecture for Satellite Image Classification

Submitted to BIM2025 Conference

Abstract

This research presents an innovative approach to satellite image classification by integrating Convolutional Block Attention Module (CBAM) with the EfficientNetB2 architecture. The lightweight model achieves superior performance in classifying satellite imagery while maintaining computational efficiency, making it suitable for real-world deployment in resource-constrained environments.

Key Contributions

  • Integration of CBAM attention mechanism with EfficientNetB2 for enhanced feature extraction
  • Lightweight architecture optimized for satellite image classification tasks
  • Improved accuracy with reduced computational requirements
  • Practical applications in remote sensing and geospatial analysis

NextFit - AI Fitness Trainer Web Application

CSE-380 Internet Programming Project

Project Screenshot

Project Overview

NextFit is a comprehensive full-stack fitness and diet planner web application that revolutionizes the way users approach their fitness journey. The platform combines modern web technologies with artificial intelligence to create an interactive and personalized fitness experience.

Key Features

  • Real-time AI Voice Assistant: Integrated voice interactions powered by Gemini AI, allowing users to get instant fitness advice, workout recommendations, and nutritional guidance through natural conversation.
  • Secure Authentication: Implemented robust user authentication using Clerk, ensuring secure login and user data protection.
  • Cloud Database: Utilized Convex for real-time cloud database management, providing seamless data synchronization and storage.
  • Dynamic Profile Dashboard: Personalized user dashboards that track progress, display workout history, and manage diet plans.
  • Responsive Design: Built with Tailwind CSS for a mobile-first, fully responsive user interface.
  • Modern Architecture: Developed with Next.js and TypeScript for type-safe, scalable, and performant application structure.

Technical Implementation

The application leverages Next.js 14 with TypeScript for robust type checking and modern React features. Clerk handles all authentication flows including sign-up, sign-in, and session management. Convex provides a real-time backend database that automatically syncs data across all clients. The AI voice assistant uses Gemini AI API for natural language processing and intelligent responses.

Deployment

The application is deployed on Vercel, taking advantage of edge functions and automatic scaling to ensure optimal performance and reliability for users worldwide.

Technology Stack
